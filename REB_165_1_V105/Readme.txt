1. Installieren Sie den USB Treiber "Treiber\MCP2200 Windows Driver 2014-10-09\Driver Installation Tool"
Im Windows Geräte Manager finden Sie nun ein neues Gerät mit einem virtuellen COM-Port
2. Installieren Sie die REB165 Software über die Setup.exe (Microsoft .NET Framework 4.0 muss installiert sein)
3. Starten Sie des REB165 Software und verbinden Sie sich mit dem richtigen COM-Port
